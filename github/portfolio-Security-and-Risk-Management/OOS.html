2FA login for Online Shopping System

I have created a 2FA program in Python that allows users to either register a new account or log in to an existing account then will receive one-time password before they are granted access. If a user chooses to register, they will be prompted to enter their desired username and password, and the program will store this information in a text file called "pass.txt". If a user chooses to log in, they will be prompted to enter their username and password, and the program will check if the entered username and password match those stored in "pass.txt". If the login details are correct, the user will be welcomed, and the program will prompt the user to enter a one-time password that was sent to “otp.txt”. The program will check if the entered password matches the one stored in “otp.txt”. If the one-time password is correct, the user will be granted access, and the program will exit. If the one-time password is incorrect, the user will be denied access, and the program will prompt the user to enter the password again.
